# colorful-noise

Research code and demo surfaces for spectral control and editing in diffusion models.

## Public pages

- Roadmap site: `docs/roadmap/index.html`
- GitHub Pages entrypoint: `docs/index.html` -> redirects to the roadmap site

## Interactive demo

The browser demo is a Gradio app in `experiments/spectral_demo.py`.

FLUX demo tabs:

```bash
uv run experiments/spectral_demo.py --model flux-dev
```

Default local URL:

```text
http://127.0.0.1:7860
```

For the full repeatable setup and GitHub Pages steps, see `docs/DEMO_SETUP.md`.
